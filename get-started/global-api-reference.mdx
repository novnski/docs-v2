Consolidated Moralis Data API Reference\
\
This is a single, consolidated quick-reference catalog of the Moralis Data API endpoints.\
\
1. _A unified API to fetch balances, transfers, DeFi positions, PnL, NFTs, and fully decoded wallet activity across multiple chains._

| Endpoint                 | Description                                    |
| :----------------------- | :--------------------------------------------- |
| **Wallet History**       | Complete decoded activity feed for a wallet.   |
| **Token Balances**       | Current ERC-20 holdings for a wallet.          |
| **NFT Balances**         | Current NFT holdings for a wallet.             |
| **Net Worth**            | Total wallet value in USD.                     |
| **Wallet P&L**           | Profit / loss calculated per token.            |
| **Decoded Transactions** | Human-readable transaction data and summaries. |

2. EVM Token API\
\
_The most powerful Token API in Web3 - fetch, analyse, and monitor ERC20 tokens across multiple chains, covering prices, balances, transfers, liquidity, holders, volume, profitability, and advanced safety signals._

| Endpoint            | Description                                          |
| :------------------ | :--------------------------------------------------- |
| **Token Balances**  | Get token holdings for any wallet.                   |
| **Token Transfers** | Get transfer history by wallet or token contract.    |
| **Token Price**     | Get real-time and historical price for a token.      |
| **Token Metadata**  | Get token details including name, symbol, and logo.  |
| **Token Search**    | Search for tokens by name or symbol.                 |
| **Token Pairs**     | Get DEX trading pairs, reserves, and liquidity data. |

3. EVM NFT API\
\
_Everything needed to build NFT experiences: metadata, ownership, transfers, sales, rarity, floor prices, and marketplace activity._

| Endpoint                | Description                           |
| :---------------------- | :------------------------------------ |
| **NFT Metadata**        | Get metadata for a specific NFT.      |
| **Collection Metadata** | Get collection-level metadata.        |
| **Floor Price**         | Current floor price for a collection. |
| **NFT Owners**          | List all owners of a collection.      |
| **NFT Transfers**       | Transfer history for a collection.    |
| **Collection Trades**   | Recent sales for a collection.        |

4. EVM DeFi API\
\
_Track DeFi positions, balances, rewards, and protocol interactions with enriched, protocol-aware data._

| Endpoint               | Description                            |
| :--------------------- | :------------------------------------- |
| **Wallet Protocols**   | Summary of protocols used by a wallet. |
| **Wallet Positions**   | Detailed positions per protocol.       |
| **Detailed Positions** | Enhanced position breakdown.           |

5. Solana API

Here is the Solana API table from the consolidated document:

| Endpoint             | Description                                                                  |
| :------------------- | :--------------------------------------------------------------------------- |
| **Native Balance**   | Gets the native SOL balance owned by the given address.                      |
| **Token Balances**   | Gets the token balances (SPL tokens) owned by the given address.             |
| **Wallet Swaps**     | Get all swap-related transactions (buy, sell) for a specific wallet address. |
| **Wallet Portfolio** | Calculates and returns the complete portfolio overview for a wallet.         |
| **NFT Balances**     | Gets the NFT balances for a given wallet address.                            |
| **Token Metadata**   | Retrieves metadata for a specific SPL token contract.                        |

6. Universal API (Overview)\
\
_The entry point to Moralis' cross-chain data, offering token and market data that is chain-agnostic._

| Endpoint               | Description                                                 |
| :--------------------- | :---------------------------------------------------------- |
| **Overview**           | High-level summary of Moralis' Universal Data capabilities. |
| **Market Metrics API** | Access to token market data like volume and liquidity.      |

7. EVM Price API\
\
_Dedicated APIs to fetch comprehensive historical and real-time token price data (separate from the Token API)._

| Endpoint              | Description                                     |
| :-------------------- | :---------------------------------------------- |
| **Overview**          | High-level summary of EVM Price API endpoints.  |
| **OHLC Candlesticks** | Get historical OHLC (Open-High-Low-Close) data. |

8. EVM Blockchain API\
\
_Core block-level data, logs, and internal transactions for EVM chains._

| Endpoint      | Description                                         |
| :------------ | :-------------------------------------------------- |
| **Overview**  | High-level summary of EVM Blockchain API endpoints. |
| **Get Block** | Retrieve a specific block by number or hash.        |
