## Overview

The **NFT API** provides complete NFT data including metadata, ownership, trading history, pricing, traits, and collection analytics across all supported EVM chains.

Whether you're building a marketplace, portfolio tracker, or analytics platform, the NFT API gives you everything you need to work with NFTs.

---

## What Is the NFT API?

The NFT API lets you query:

- **Metadata** - Token and collection metadata with media URLs and IPFS resolution
- **Ownership** - Current owners, ownership history, and holder counts
- **Transfers** - Transfer history by collection, token ID, or wallet
- **Pricing** - Floor prices, sale history, and price trends
- **Traits** - Attribute data, trait counts, and rarity analysis
- **Discovery** - Top collections by volume, market cap, or activity

---

## Key Features

- **Rich Metadata** - Fully resolved metadata with IPFS gateway support
- **Floor Price Tracking** - Current and historical floor prices per collection
- **Sales Data** - Trade history with prices, marketplaces, and transaction details
- **Trait Analysis** - Collection traits with rarity percentages and counts
- **Ownership Queries** - Find all owners or check ownership by wallet
- **Market Discovery** - Discover trending collections by various metrics
- **Batch Queries** - Fetch metadata for multiple NFTs in a single request
- **Multi-Chain** - Consistent data across Ethereum, Polygon, Base, and more

---

## Common Use Cases

- **NFT Marketplaces**\
  (listings, metadata display, ownership verification)
- **Portfolio Trackers**\
  (display NFT holdings with floor price valuations)
- **Analytics Platforms**\
  (collection stats, volume trends, floor price charts)
- **Rarity Tools**\
  (trait analysis and rarity rankings)
- **Token Gating**\
  (verify NFT ownership for access control)
- **Collection Dashboards**\
  (monitor collection performance and activity)
