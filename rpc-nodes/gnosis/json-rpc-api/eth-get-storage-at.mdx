---
title: "eth_getStorageAt"
description: "Returns the value from a storage position at a given address."
---

#### Path parameters

<ResponseField name="apiKey" type="string" required>
  Your API key for authentication
</ResponseField>

#### Request

<ResponseField name="address" type="string" required>
  Address of the storage
</ResponseField>

<ResponseField name="position" type="string" required>
  Integer of the position in the storage
</ResponseField>

<ResponseField name="tag" type="string">
  Tag for the state ("latest", "earliest", "pending"). Defaults to "latest" if null
</ResponseField>

#### Example request `params`

```javascript
[
  "0x295a70b2de5e3953354a6a8344e616ed314d7251",
  "0x0",
  "latest"
]
```

<RequestExample>

```shellscript curl
curl https://site1.moralis-nodes.com/gnosis/:apiKey \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc": "2.0", "method": "eth_getStorageAt", "params": ["0x295a70b2de5e3953354a6a8344e616ed314d7251", "0x0", "latest"], "id": 1}'
```

</RequestExample>

<ResponseExample>

```json JSON
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x00000000000000000000000000000000000000000000000000000000000004d2"
}
```

</ResponseExample>
