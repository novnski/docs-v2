title: "eth_blockNumber"
description: "Returns the latest block number of the blockchain."
---

#### Path parameters

<ResponseField name="chain" type="string" default="eth" required>
  The blockchain to query
</ResponseField>

<ResponseField name="apiKey" type="string" required>
  Your API key for authentication
</ResponseField>

<RequestExample>

```shellscript curl
curl https://site1.moralis-nodes.com/eth/:apiKey \
  -X POST \
  -H "Content-Type: application/json" \
  --data '{"method":"eth_blockNumber","params":[],"id":1,"jsonrpc":"2.0"}'
```

</RequestExample>

<ResponseExample>

```json JSON
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x15536ee"
}
```

</ResponseExample>
